# Run in Development

1. Clone the repository
2. Run
```
yarn install
```

3. Make sure you have Nest CLI installed
```
npm i -g @nestjs/cli
```

4. Start the database
```
docker-compose up -d
```

## Stack used
* NestJS
* MongoDB
